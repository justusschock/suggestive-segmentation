# Suggestive Segmentation

## Content
This repository contains an early stage of suggestive segmentation using a u-net. It has been abandoned in favor of more accurate segmentation approaches.

## Requirements: 
* python 3.5
* pytorch v 0.2
* torchvision v 0.1.9
* pillow v 4.1.2
* numpy v 1.11.3
* yattag v 1.9.2
* yaml v 0.1.6
* matplotlib v 1.5.1
